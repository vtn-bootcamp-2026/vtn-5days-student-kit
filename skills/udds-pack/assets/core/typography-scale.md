# Typography Scale

This file defines the scale ratios and naming conventions for typography across all themes. It does not assign specific font families.

- **H1 (Hero/Title):** Used for main slide titles and hero sections. Bold, large, clear.
- **H2 (Subtitle/Section):** Used for section dividers and framing lines. Medium weight.
- **Body:** Used for general content and bullet points (max 3-5 lines per slide). Spacious, readable.
- **Caption/Label:** Used for small footer text, chart labels, and minor annotations. Small, discreet.

*Note: Vietnamese titles should follow sentence case (e.g. "Xây dựng năng lực AI cho tổ chức"). Avoid all-caps for long text.*
