# Spacing Scale

This file defines the universal spacing rules (padding and margins) using T-shirt sizing to maintain consistent rhythm across all layouts.

- **xs (Extra Small):** Minimal padding between tightly coupled elements (e.g., icon and label).
- **sm (Small):** Padding within small cards or between list items.
- **md (Medium):** Standard padding within main content blocks and sections.
- **lg (Large):** Margin between major layout columns or large distinct sections.
- **xl (Extra Large):** Hero spacing, large section divider padding, and outer slide margins.

*Note: Slides should adhere to a golden ratio margin approach to ensure content breathes and feels boardroom-ready.*
